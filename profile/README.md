## 🧠 What We Study

다음과 같은 CS 핵심 주제를 중심으로 스터디를 진행합니다.

- Computer Architecture
- Operating System
- Data Structure
- Network
- Database

각 주제는 **발표 → 질문 → 토론 → 정리**의 과정을 거쳐  
노션과 GitHub에 구조화된 형태로 기록됩니다.

## 📝 How We Learn & Record

- 발표자는 **주제 정리 + 발표**
- 참여자는 **사전 학습 + 질문**
- 스터디 이후  
  - 논의된 내용 반영
  - 정제된 형태로 **GitHub Repository에 업로드**

> 모든 기록은  
> 👉 *“나중에 다시 봐도 이해되는 수준”* 을 기준으로 합니다.
